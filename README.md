## Что умеет
- Показывать расписание, оценки и д\з
- Отображает за что выставлена оценка
- Умеет считать, какая оценка будет, если к текущей добавить 2,3,4 или 5

- Retrofit
- Gson
- Hilt
- Firebase
- Jsoup
- Markwon
- Glide
- Shimmer
- И чуть чуть Jetpack Compose
